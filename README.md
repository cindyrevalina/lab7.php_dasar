## Deskripsi
Tugas ini merupakan latihan membuat program **PHP dasar** menggunakan form input untuk menampilkan:
- Nama
- Tanggal lahir
- Pekerjaan  
Program ini juga menghitung **umur otomatis** berdasarkan tanggal lahir dan menampilkan **gaji** sesuai pilihan pekerjaan.

---

## Tujuan Praktikum
Praktikum ini bertujuan untuk:
1. Mengenal dan memahami dasar-dasar bahasa pemrograman **PHP**.
2. Mempelajari cara kerja **server-side scripting** menggunakan PHP di lingkungan **localhost (XAMPP)**.
3. Menerapkan konsep **variabel**, **operator**, **struktur kondisi (if, switch)**, dan **perulangan (looping)**.
4. Membuat form input sederhana untuk menghitung **umur** dan menampilkan hasil sesuai dengan **pekerjaan yang dipilih**.
